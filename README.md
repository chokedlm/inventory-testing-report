# QA Portfolio

## RU
**Inventory** - тестовое приложение, написанное на Python для отработки тестирования и написания тестовой документации. Приложение включает в себя функции:
- Открыть инвентарь
- Купить предмет
- Экипировать предмет 
- Удалить предмет
- Закрыть инвентарь

Документация включает в себя:
- План тестирования
- Чек-лист тестирования
- Таблицу тест-кейсов
- Таблицу баг-репортов
- Отчет о тестировании

### Как запустить:
#### Первый способ: запуск из исходного кода
- Скачайте репозиторий
```bash
git clone https://github.com/chokedlm/inventory-testing-report
```
- Перейдите в папку проекта
```bash
cd inventory-testing-report
```
- Запустите код
```bash
python inventory.py
```
